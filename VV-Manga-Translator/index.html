Privacy Policy

Last updated: January 2025

VV Manga Translator (“the App”) is designed as a privacy-first manga translation tool. This Privacy Policy explains how the App handles your data and the safeguards applied to keep your information secure.

Data Collection & Storage

The App does not collect, transmit, or sell personal information. All content remains on your device.

Specifically:

  • Manga images, translation results, and project metadata are stored locally on your device only.
  • No personal data is uploaded to servers or synced with cloud services.
  • The App does not require or offer account registration or user profiles.
  • There is no built-in cloud backup; all processing occurs offline unless you request a translation from a third-party provider.

Network Usage

The App operates offline by default. Network access is used solely for outbound translation requests when you choose an online engine. These requests are limited to recognized Japanese text extracted from the selected manga page. Images or other media are never uploaded.

Third-Party Services

If you enable online translation, the App can connect to the following services (subject to the API keys you configure):

  • Google Translate
  • DeepSeek API
  • OpenAI ChatGPT API

Each provider processes only the text you send and returns translation results. Please review their respective privacy policies for additional details. By using these services, you agree to their terms and data practices.

Data Security

We implement the following safeguards to help protect your information:

  • All data is written to local storage under your control.
  • Removing the App does not automatically delete saved translations; you must delete the files manually if desired.
  • We recommend creating regular backups of important translations if you plan to reinstall or migrate devices.
  • API keys are supplied by you (or the developer build configuration) and stored securely within the App’s configuration; they are not shared with third parties beyond the selected translation service.

Permissions

The App requests the minimal set of runtime permissions required for its core functionality:

  • Storage – access manga image files and store translation cache data.
  • Network – send text segments to online translation providers when you opt to use them.

User Controls

  • You can clear all cached translations from the Settings screen at any time.
  • Switching to the “Offline OCR” workflow ensures no network requests are issued.
  • You may choose alternative translation engines or disable them entirely according to your privacy preferences.

Children’s Privacy

The App targets a general audience and does not knowingly collect personal data from children. Because no personal data is collected, parental consent frameworks are not required.

Contact

If you have privacy-related questions or requests, please contact the project maintainers via the official repository issue tracker.

This Privacy Policy may be updated to reflect product changes or legal requirements. Continued use of the App after updates constitutes acceptance of the revised policy.
